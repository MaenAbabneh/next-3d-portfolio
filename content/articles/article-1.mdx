Three.js يسهّل استخدام WebGL بشكل عملي داخل الويب. الفكرة الأساسية: **مشهد (Scene)** يحتوي عناصر، **كاميرا (Camera)** تحدد منظورك، و**Renderer** يرسم النتيجة.

<Note title="ملاحظة مهمة">
  كل عنصر في المشهد (Scene) له دور محدد. لا تخلط بين الكاميرا (Camera) والمجسمات
  (Mesh) — الكاميرا فقط تحدد منظور الرؤية.
</Note>

## Minimal mental model

- Scene: صندوق يجمع كل شيء
- Camera: العين
- Mesh: شكل + خامة
- Light: يغير الشكل والإحساس

tick();

<Example title="مثال عملي">
  ```ts
  // pseudo-code (concept only)
  const scene = new Scene();
  const camera = new PerspectiveCamera();
  const renderer = new WebGLRenderer();

const cube = new Mesh(geometry, material);
scene.add(cube);

function tick() {
cube.rotation.y += 0.01;
renderer.render(scene, camera);
requestAnimationFrame(tick);
}

tick();

```
